## 🔧 安装和测试指南

### 1. 环境要求
- Python 3.9+
- Windows 10+/Linux/macOS
- 已安装OpenClaw

### 2. 安装步骤
```bash
# Windows
1. 下载并解压
2. 以管理员身份运行 install_windows.bat
3. 检查依赖是否安装成功

# Linux
1. 下载并解压
2. chmod +x install_linux_mac.sh
3. ./install_linux_mac.sh
4. 检查依赖安装

# macOS
1. 下载DMG文件
2. 拖拽到应用程序
3. 可能需要安全设置允许
```

### 3. 测试项目
请测试以下功能并报告结果：

#### 核心功能测试
- [ ] **GUI启动**: 程序是否能正常启动？
- [ ] **悬浮窗显示**: 是否能正常显示和拖拽？
- [ ] **安全扫描**: 点击"扫描风险"按钮是否工作？
- [ ] **报告生成**: 是否能生成安全报告？
- [ ] **Token显示**: Token计数器是否更新？

#### 兼容性测试
- [ ] **Windows兼容性**: 在Windows系统上是否正常？
- [ ] **Linux兼容性**: 在Linux系统上是否正常？
- [ ] **macOS兼容性**: 在macOS系统上是否正常？
- [ ] **OpenClaw版本**: 与你当前的OpenClaw版本是否兼容？

#### 稳定性测试
- [ ] **长时间运行**: 运行1小时是否稳定？
- [ ] **多任务处理**: 同时进行多个操作是否正常？
- [ ] **错误恢复**: 遇到错误后是否能恢复？
- [ ] **资源使用**: CPU和内存使用是否合理？
